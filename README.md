This is an example repository that illustrates how to configure `@dependabot` to automate the process of upgrading OpenJDK in Docker images. It restricts the upgrades to the current major version.

For a more detailed description of how this works, you can checkout [this article](https://medium.com/devops-by-nature/how-to-configure-dependabot-to-automatically-upgrade-openjdk-in-docker-images-7738109e9506?sk=7dd594717fcf7f815791cc7db26e72d0).
